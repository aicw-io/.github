### Open Source Tools

#### [Ask AI Widget](https://github.com/aicw-io/aicw-widget-ask-ai)

Floating button that lets visitors summarize any page with **ChatGPT, Claude, Perplexity, Gemini, or Grok** — and share it via WhatsApp, X, LinkedIn, and more. Configurable colors, position, and services. No tracking, no analytics. MIT License.

```html
<script src="https://cdn.jsdelivr.net/gh/aicw-io/aicw-widget-ask-ai@latest/dist/aicw-widget-ask-ai.min.js"></script>
```

#### [Params Saver](https://github.com/aicw-io/aicw-params-saver)

Captures UTM and marketing parameters, stores them in the browser, and decorates internal links — so attribution survives the full user journey. GDPR-compliant by default, SPA-ready, no cookies. MIT License.

```html
<script src="https://cdn.jsdelivr.net/gh/aicw-io/aicw-params-saver@latest/dist/aicw-params-saver.min.js"></script>
```
